此資料夾內有不同模型訓練後的權重檔案和分數紀錄
### datalist.pkl -- 一個pkl檔案，打開後是一個array形式的資料。
* 第0項是該模型最後儲存的記憶
* 第1項是當時的epsilon值
* 第2項是訓練的局數
* 第3項是分數記錄列表
  * 第[3][n][0]項是記錄第n局的最終分數
  * 第[3][n][1]項是記錄第n局的合成出的最大格子值
  * 第[3][n][2]項是記錄第n局的最終移動次數

### 2048_weights.h5 -- 儲存成h5格式的權重檔案，好像沒啥好說的。
